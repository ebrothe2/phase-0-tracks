Reflect

Semantic tags give meaning to different sections of the web page. A way of breaking up the content in a logical way. Currently I see the aside and nav as most useful since they seem to have fairly specific uses. In this challenge I learned about Snippets in sublime. I will definitely practice on making a snippet again just to get more comfortable with how to make and use this feature. For me the most confusing part on this challenge was decifering the difference between some of the semantic tags that seemed very similar - like the difference between main and body. I think I will take some time to look at some page source codes to see how they use semantic tags to get some clarification.
